# 构建 Python 编译环境

创建 virtual env，然后用 `maturin develop` 构建：

```bash
- cd queryer-py
- python -m venv .env
- source .env/bin/activate
- window: .\.env\Scripts\Activate.ps1
- pip install maturin
- maturin develop
```
